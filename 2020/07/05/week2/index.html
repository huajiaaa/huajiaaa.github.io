Using Letsencrypt with the node exporter

众所周知, Prometheus 可以从 node exporter 中拉取监控数据。node exporter 在 1.0 版本开始对安全传输层协议（TLS）有了支持, 可以连接到Letsencrypt（证书颁发, 开源且免费, 其颁发的证书几乎已经被所有的浏览器认可）。

node exporter 中并没有内置支持证书的续订, 而是如果发现证书在磁盘上有了修改, 则自动开始使用新的证书, 因此可以使用 Cerbot（Letsencrypt 的官方客户端） 实现自动续订证书。然后文章演示了一下如何使用…
